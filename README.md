# Task Manager

A simple Task Manager application built using HTML, CSS, and JavaScript.

## Features

- Add Tasks
- Complete Tasks
- Delete Tasks
- Add Task using Enter Key
- Clean User Interface

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

1. Download the project
2. Open index.html in a browser
3. Start managing your tasks
